# Line get Information to Web Storage

หน้าแสดงข้อมูลฟอร์ม และนำ Line Login มาใช้เพื่อ get Infomation จากไลน์ แล้วจัดเก็บผ่าน Web Storage


## Installation

Use Google Firebase

```bash
firebase emulators:start // สำหรับ Run on Localhost
```
